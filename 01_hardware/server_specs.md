# Server Specifications

## System Overview

- Manufacturer: HP
- Model: ProLiant Microserver Gen8
- Intended role:
 - Acts as a self-hosted infrastructure platform for cloud photo storage service ("MediaSafe-Cloud").
 - Provides persistent, remotely accessible services without requiring direct physical interaction with the server.
 - Serves as a learning environment for operating systems, storage, networking and security fundamentals. 

 ## CPU

- Model: Intel Xeon E3-1280
- Architecture: x64
- Cores / Threads: 4 cores, 8 threads
- Virtualization support: Intel® Virtualization Technology (VT-x)

## Memory

- Total Installed: 16GB RAM installed. 
- Type: DDR3
- ECC Support: Yes
- Slot configuration: Dual-channel mode
- Maximum supported: 16GB RAM (platform limitation)

## Storage

| Disk | Type | Capacity | Interface | Intended Use |
|-----|------|----------|-----------|--------------|
|   Disk 1  |  SSD (Crucial)    | 500GB | SATA | Operating System and cloud photo application |
|   Disk 2  |  HDD (Seagate Barracuda) | 2TB | SATA | Primary photo and video data |
|   Disk 3  |  HDD (Seagate Barracuda) | 2TB | SATA | Primary photo and video data |


## Networking

- Primary NIC: Onboard Gigabit Ethernet. 
- Secondary NIC: TP-Link Archer T3U USB WiFi Adapter
  - Interface: USB 3.0
  - Chipset: Realtek
  - Intended use: Provide network connectivity for the cloud photo storage environment.
